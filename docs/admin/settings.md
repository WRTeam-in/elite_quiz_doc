# Settings

The Settings section in the Elite Quiz Admin Panel provides comprehensive control over the appearance, behavior, and functionality of your quiz application.

## General Settings

### Basic Application Settings

1. Navigate to **Settings > General Settings**
2. Configure the following options:
   - Application Name
   - Admin Email
   - Default Language
   - Default Time Zone
   - Date Format
   - Site Logo
   - Favicon
   - Copyright Text
3. Click **Save Changes** to apply your settings

## Theme Settings

### Appearance Customization

1. Navigate to **Settings > Theme Settings**
2. Customize the visual appearance:
   - Primary Color
   - Secondary Color
   - Font Family
   - Button Styles
   - Card Styles
   - Header/Footer Layout
3. Click **Save Theme** to apply your changes

### Custom CSS

1. In the Theme Settings, go to the **Custom CSS** tab
2. Add your own CSS code to customize the appearance further
3. Use the live preview to see changes before saving
4. Click **Save Custom CSS**

## Language Settings

### Managing Languages

1. Navigate to **Settings > Language Settings**
2. Here you can:
   - Add new languages
   - Edit existing languages
   - Set default language
   - Enable/disable languages
   - Manage translations

### Adding a New Language

1. Click **Add Language**
2. Enter the language details:
   - Language Name
   - Language Code
   - Flag Icon
   - RTL Support (Yes/No)
3. Click **Add Language**

### Managing Translations

1. Select a language from the list
2. Click **Manage Translations**
3. Edit translations for all system texts
4. Click **Save Translations**

## Email Settings

### Email Configuration

1. Navigate to **Settings > Email Settings**
2. Configure your email settings:
   - SMTP Host
   - SMTP Port
   - SMTP Username
   - SMTP Password
   - Encryption Type (TLS/SSL)
   - From Email
   - From Name
3. Click **Save Email Settings**

### Email Templates

1. Navigate to **Settings > Email Templates**
2. Edit the various email templates:
   - Welcome Email
   - Password Reset
   - Verification Email
   - Quiz Completion
   - Certificate Delivery
3. Use template variables like `{{user_name}}`, `{{quiz_title}}`, etc.
4. Click **Save Template** for each template you modify

## Security Settings

### Security Configuration

1. Navigate to **Settings > Security Settings**
2. Configure security parameters:
   - Password Complexity Rules
   - Account Lockout Settings
   - Session Timeout
   - Two-Factor Authentication Options
   - CAPTCHA Settings
3. Click **Save Security Settings**

### API Security

1. Navigate to **Settings > API Settings**
2. Configure API access:
   - Enable/Disable API Access
   - API Rate Limiting
   - API Key Management
   - IP Restrictions
3. Click **Save API Settings**

## Notification Settings

### Push Notifications

1. Navigate to **Settings > Notification Settings**
2. Configure push notification settings:
   - Enable/Disable Push Notifications
   - Firebase Configuration
   - Default Notification Sounds
   - Notification Icons
3. Click **Save Notification Settings**

### In-App Notifications

1. In the Notification Settings, go to the **In-App** tab
2. Configure settings for in-app notifications:
   - Display Duration
   - Position (Top, Bottom, Center)
   - Animation
   - Max Number of Notifications
3. Click **Save In-App Settings**

## Social Media Settings

### Social Media Integration

1. Navigate to **Settings > Social Media Settings**
2. Enter your social media links:
   - Facebook
   - Twitter
   - Instagram
   - YouTube
   - LinkedIn
3. Configure social sharing options:
   - Default Share Messages
   - Share Image
   - Share Buttons Display
4. Click **Save Social Media Settings**

## Payment Settings

### Payment Gateway Configuration

1. Navigate to **Settings > Payment Settings**
2. Configure payment gateways:
   - Stripe
   - PayPal
   - Razorpay
   - Other Supported Gateways
3. Set transaction fees, currencies, and payment options
4. Click **Save Payment Settings**

## Backup Settings

### Database Backup

1. Navigate to **Settings > Backup Settings**
2. Configure automatic backup:
   - Backup Frequency
   - Backup Storage Location
   - Retention Period
   - Notification on Backup Completion
3. Click **Save Backup Settings**

### Manual Backup

1. Navigate to **Settings > Backup Settings**
2. Click **Create Backup Now** to manually generate a backup
3. Download the backup file for safekeeping

## System Settings

### Server Configuration

1. Navigate to **Settings > System Settings**
2. View and configure system information:
   - PHP Version
   - MySQL Version
   - Server Information
   - File Upload Limits
   - Memory Limits
3. Click **Save System Settings**

### Maintenance Mode

1. Navigate to **Settings > Maintenance Mode**
2. Enable or disable maintenance mode
3. Configure maintenance page message
4. Set allowed IP addresses that can access the site during maintenance
5. Click **Save Maintenance Settings**
