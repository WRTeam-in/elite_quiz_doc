# Features

Elite Quiz offers a comprehensive set of features designed to create engaging and interactive quiz experiences. This guide provides detailed information about all available features and how to use them effectively.

## Quiz Creation

### Question Types

- **Multiple Choice**

  - Single correct answer
  - Multiple correct answers
  - Image-based questions
  - Audio-based questions
  - Video-based questions

- **True/False**

  - Simple binary questions
  - Explanation support
  - Image support
  - Media integration

- **Fill in the Blanks**

  - Text-based answers
  - Case sensitivity options
  - Multiple blanks
  - Partial credit support

- **Matching**

  - Drag and drop interface
  - Multiple pairs
  - Image matching
  - Text matching

- **Short Answer**

  - Text input
  - Word limit
  - Character limit
  - Auto-grading options

- **Long Answer**
  - Essay questions
  - Rich text support
  - Manual grading
  - Rubric support

### Quiz Settings

- **Timing**

  - Overall time limit
  - Per-question time limit
  - Time warnings
  - Auto-submission

- **Scoring**

  - Point system
  - Negative marking
  - Partial credit
  - Custom scoring rules

- **Display**

  - Question order
  - Option order
  - Progress tracking
  - Results display

- **Access**
  - Password protection
  - Schedule availability
  - IP restrictions
  - User restrictions
