# Features Overview

Elite Quiz offers a comprehensive set of features designed to create engaging quiz experiences for your users. This section provides detailed explanations of key features to help you make the most of the platform.

---

## Quiz Types

Elite Quiz supports multiple quiz types to suit different use cases and engagement styles:

### Available Quiz Types

1. **Audio Quiz** - Audio-based questions for listening comprehension and pronunciation
2. **Contest** - Competitive quiz mode with rankings and prizes
3. **Exam** - Formal assessment mode for testing and certification
4. **Fun and Learn** - Educational quizzes with an entertaining approach
5. **Guess the Word** - Word puzzle and vocabulary building quizzes
6. **Maths Quiz** - Mathematical problem-solving challenges
7. **Multi Match** - Questions with multiple correct answers and/or correct order sequencing (can be mixed)
8. **Quiz Zone** - Comprehensive quiz mode with multiple sub-types:
   - **Bookmark** - Save and revisit favorite questions
   - **Daily Quiz** - New quiz challenges every day
   - **Group Battle** - Compete with multiple players simultaneously (max 4 users)
   - **One vs. One Battle** - Head-to-head competition between two players
   - **Random Battle** - Quick match with random opponents (1v1)
   - **Self Challenge** - Practice mode for self-improvement
   - **True or False** - Quick binary choice questions
